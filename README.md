# Java Basics

A collection of beginner Java programs.


---

## Projects

### 🚗 CarLoan
Calculates the estimated monthly car payment based on loan amount, interest rate, loan length, and down payment.
- **Concepts:** variables and conditionals: if/else

### 🌍 Continents
Prints the name of a continent and its largest city based on a number input.
- **Concepts:** switch statements

### 🤖 Droid
Simulates a droid object with a name, battery level, and tasks it can perform.
- **Concepts:** classes, constructors, instance variables and methods

### 🔢 Calculator
A simple arithmetic calculator that can add, subtract, multiply, divide and find remainders.
- **Concepts:** classes, constructors, methods and return values

### 🎵 Playlist
Manages a music playlist using arrays and ArrayLists - including adding, removing, and swapping songs.
- **Concepts:** arrays, ArrayList, add, remove, indexOf and set statements

### 🔭 PrimeDirective
A collection of number-based methods including prime checking, parity filtering, and Fibonacci sequence generation.
- **Concepts:** methods, ArrayLists, for each loops and boolean return values
